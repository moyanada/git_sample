## ✅ **GIT_SAMPLE** ##

## ✅ 설명
- Git 명령어 테스트용

## ✅ 명령어
```
초기
git init
git remote add origin https://github.com/moyanada/git_sample.git
git add .
git commit -m "frist"
git push -u origin master

로컬에서 돌아가기 (hard, soft, mixed)
git reset --hard {commit_id}

Merge 되돌리기
git revert -m 1 {commit_id}
```
